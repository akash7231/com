-- CAPABILITIES OF SELECT

1. SELECTION - select one or more rows with all columns
2. PROJECTION - select one or more rows with specific columns
3. JOIN - select rows combined from two or more tables

SQL is a non procedural language

SELECT * FROM EMP;
